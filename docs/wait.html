wait

wait

Creates a new stream with a single value that's a Buffer of the entire contents of the stream.
Since:
  • 1.0.0
Source:
  • consumers/v2/wait.js, line 39

Example

// get the contents of a file
fs.createReadStream('example.txt')
  .pipe(nodeStream.wait());
  // => Buffer
